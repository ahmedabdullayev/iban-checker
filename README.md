# Documentation

### Technologies
- Vue 3
- TypeScript

### How to use?
There is two pages, one is for single IBAN input, second is for uploading text file with IBANs(one IBAN per line in .txt file)

### Features

- Check IBANs if they are correct
- Single IBAN check
- List of IBANs check
